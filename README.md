# rel!c Landing Page

Одностраничный сайт для продажи скульптуры "Голова Будды".
Сайт выполнен в стиле "Музейный минимализм" с использованием React, TypeScript, Vite и TailwindCSS.

## Запуск

1. **Установка зависимостей** (если вы скопировали проект):
   ```bash
   npm install
   ```

2. **Запуск локального сервера**:
   ```bash
   npm run dev
   ```
   Сайт будет доступен по адресу: `http://localhost:5173` (или 5174, если порт занят).

## Структура

- `src/components`: Компоненты для каждого блока (Hero, Manifest, Product, InSitu, Partners, Contacts).
- `src/assets`: Изображения (сгенерированные заглушки).
- `tailwind.config.js`: Настройки стилей (шрифты, цвета).

## Как заменить фото

1. Положите ваши настоящие фото в папку `src/assets`.
2. Откройте `src/App.tsx`.
3. Замените импорты в начале файла:
   ```tsx
   import heroImg from './assets/ВАШЕ_ФОТО_HERO.jpg';
   import textureImg from './assets/ВАШЕ_ФОТО_ТЕКСТУРЫ.jpg';
   // ...
   ```
   Или переименуйте ваши фото в имена, которые уже используются (`buddha_head_hero.png` и т.д.).

## Сборка для публикации

```bash
npm run build
```
Файлы для загрузки на хостинг появятся в папке `dist`.
